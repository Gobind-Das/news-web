# Bengali News Article Portal

একটি সম্পূর্ণ প্রস্তুত Django-ভিত্তিক বাঙা সংবাদ পোর্টাল।

## বৈশিষ্ট্য

✅ **ডেস্কটপ & মোবাইল রেসপন্সিভ ডিজাইন**
- শীর্ষ নেভিগেশন বার
- মোবাইল হ্যাম্বারগার মেনু

✅ **11টি প্রি-কনফিগার করা বিভাগ**
- পশ্চিমবঙ্গ, ভারত, আন্তর্জাতিক, ইতিহাস, সংস্কৃতি
- ধর্ম কথা, খেলাধুলো, এবিভিপি, সংঘ, বিনোদন
- বিমর্শ, সম্পাদকীয়

✅ **সিকিউর অ্যাডমিন প্যানেল**
- শুধুমাত্র প্রামাণিক ব্যবহারকারীরা অ্যাক্সেস করতে পারে
- সম্পূর্ণ অনুমতি পরিচালনা

✅ **আর্টিকেল ম্যানেজমেন্ট**
- TinyMCE ওয়ার্ডপ্রেস-স্টাইল রিচ টেক্সট এডিটর
- ছবি আপলোড সমর্থন
- ড্রাফ্ট/প্রকাশিত/আর্কাইভ স্ট্যাটাস

✅ **SEO অপ্টিমাইজেশন**
- কাস্টম মেটা শিরোনাম & বর্ণনা
- স্লাগ-ভিত্তিক URL
- কীওয়ার্ড সমর্থন
- সাইটম্যাপ-প্রস্তুত

✅ **ব্যবহারকারী বান্ধব ফিচার**
- অনুসন্ধান কার্যকারিতা
- সম্পর্কিত আর্টিকেল
- দেখার গণনা ট্র্যাকিং
- পেজিনেশন

## ইনস্টলেশন

```bash
# ভার্চুয়াল পরিবেশ তৈরি করুন
python3 -m venv venv
source venv/bin/activate

# প্যাকেজ ইনস্টল করুন
pip install -r requirements.txt

# ডাটাবেস মাইগ্রেশন করুন
python manage.py migrate

# একজন সুপারইউজার তৈরি করুন
python manage.py createsuperuser

# ডেভেলপমেন্ট সার্ভার চালু করুন
python manage.py runserver
```

1. অ্যাডমিন প্যানেলে যান
2. "নিবন্ধ যোগ করুন" ক্লিক করুন
3. শিরোনাম, বিভাগ এবং সামগ্রী পূরণ করুন
4. বৈশিষ্ট্যযুক্ত ছবি আপলোড করুন

6. স্ট্যাটাস "প্রকাশিত" এ সেট করুন

## প্রোডাকশন স্থাপনা

প্রোডাকশনের জন্য:

```bash
# settings.py আপডেট করুন
DEBUG = False
SECRET_KEY = 'আপনার-সুরক্ষিত-কী'
ALLOWED_HOSTS = ['yourdomain.com']

# স্ট্যাটিক ফাইল সংগ্রহ করুন
python manage.py collectstatic

# Gunicorn সহ চালু করুন
gunicorn config.wsgi:application
```

## ডেটাবেস

স্থানীয় উন্নয়নের জন্য SQLite। প্রোডাকশনের জন্য PostgreSQL ব্যবহার করুন:

```python
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.postgresql',
        'NAME': 'bangavideo',
        'USER': 'postgres',
        'PASSWORD': 'password',
        'HOST': 'localhost',
        'PORT': '5432',
    }
}
```
